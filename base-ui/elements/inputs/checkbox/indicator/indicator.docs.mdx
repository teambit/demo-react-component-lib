import { CheckboxIndicator } from './indicator';

Easy styling for the native `<input type="checkbox"/>` tag, using the `input +` css selector.

Default styles include the following attributes:

- checked / unchecked
- disabled
- focus

Use the `className` prop to replace the default styles.

#### Using the indicator component

```jsx live
() => {
  return (
    <div>
      <input type="checkbox" /> → <CheckboxIndicator />
      <br />
      <input type="checkbox" defaultChecked /> → <CheckboxIndicator />
    </div>
  );
};
```
